2022

Virtual trainer
for upper limb mechanical prosthetic

The virtual trainer is a system that uses virtual reality (VR) technologies to simulate a mechanical transhumeral upper limb prosthesis and its control methods for users who need them. The system is meant to help users learn how to use the prosthesis in a realistic and immersive environment, and to select the most suitable device for their preferences and capabilities. The virtual trainer also provides feedback on the user performance and progress, which is vital to improve their motivation and confidence.
